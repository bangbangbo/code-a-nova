## ğŸ› ï¸ Quick Start Guide

### Level-1 Projects (No Installation Required)

1. Navigate to the project folder
2. Open `index.html` in your browser
3. Start using the application!

**For Weather App:**
- Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
- Add your API key to `script.js`

### Level-2 Project (Requires Node.js)

```bash
# Navigate to the project
cd Level-2/ai-trip-planner

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Add your Gemini API key to .env
# VITE_GEMINI_API_KEY=your_api_key_here

# Run the development server
npm run dev
```

## ğŸŒ Deployment Guide

### Netlify Deployment

**For Level-1 Projects:**
1. Go to [Netlify](https://www.netlify.com/)
2. Drag and drop the project folder
3. Your site is live instantly!

**For Level-2 Project:**
1. Connect your GitHub repository
2. Build command: `npm run build`
3. Publish directory: `dist`
4. Add environment variables in Site Settings
5. Deploy!

### Vercel Deployment

**For All Projects:**
1. Go to [Vercel](https://vercel.com/)
2. Import your GitHub repository
3. Select the project folder
4. Configure build settings (for React project)
5. Add environment variables
6. Deploy!

### GitHub Pages (Level-1 Only)

```bash
# Push code to GitHub
git push origin main

# Go to repository Settings > Pages
# Select source branch: main
# Select folder: / (root)
# Save and wait for deployment
```
